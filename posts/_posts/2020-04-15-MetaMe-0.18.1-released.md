This release contains bug fixes.

### Updating

You can update to this version by going to settings > update. You can also download the latest version [here](/download.html).

### Changes

##### Changed
- Installer now installs MetaMe on a per-machine level
- Installation path moved to "Program Files (x86)"
- Upgrading now requires administrator password 
- MetaMe API now exposed via http://localhost/Temporary_Listen_Addresses/MetaMe

##### Fixed

- Fixed issue where MetaMe.WindowsClient.exe is not starting under non-administrator accounts
- Fixed potential port conflict inside MetaMe.WindowsClient.exe
- Fixed logging bug affecting some people
- Fixed installer dependency on microsoft site
- Fixed installer issues when installing on non-administrator accounts
